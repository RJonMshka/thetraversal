Publicis Sapient was where production engineering became real. Four years across multiple client engagements — e-commerce platforms, financial applications, and internal tooling — working as Associate then promoted to Senior Associate.

The technical range was wide by design: transformed a financial application's data layer from REST to GraphQL, cutting network overhead significantly. Simplified AWS Lambda functions to support async/await patterns, added CloudWatch logging, and improved CI/CD pipelines. Built NLP-based search and cache-backed autocomplete. Engineered a personalized recommendation system: React frontend, MongoDB plus ElasticSearch via Node.js.

On the frontend architecture side: modularized SFCC's SFRA front-end to enable code-splitting and reduce CSS/JS co-dependencies. Achieved 30–50% decrease in load time through global lazy loading and custom caching strategies. Enforced semantic HTML and ARIA attributes for WCAG compliance across client projects.

Also mentored junior developers and delivered internal seminars on web development advancements. The mentoring instinct started here.
