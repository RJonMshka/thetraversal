Clause is a functional, English-like programming language that compiles to JavaScript. The entire toolchain is built from scratch in Go: lexer, parser, intermediate representation, runtime libraries, and code generator.

The goal is a language with the clarity of English, the structure of a functional language, and JavaScript as its compile target. This is where compiler theory meets language design philosophy.

The pipeline: source text → lexer (token stream) → parser (AST) → IR → code generation → JavaScript output. Each stage is a self-contained transformation. The parser produces a real AST — the same concept that powers this portfolio's structure.

This project is the direct successor to a Scala DSL built during the MS CS program at UIC. The same questions about language design, evaluation order, and what "meaning" means in a formal system are still alive in the current work.

*"If a language is how you think, Clause is an experiment in thinking differently."*
